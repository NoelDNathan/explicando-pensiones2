# 2026-05-27 - Fuentes oficiales para tasa de paro

- Fecha: 2026-05-27
- Objetivo: localizar fuentes oficiales para la tasa de paro en Espana desde 1975 hasta la actualidad.
- Archivos modificados: `ai/current.md`, `ai/history/2026-05-27-fuentes-tasa-paro.md`.
- Resumen de cambios: se identifico el INE EPA como fuente primaria. La EPA existe desde 1964 y es trimestral desde 1975, pero el propio INE indica que las bases de datos online difunden series trimestrales desde 1976T3 y que desde 1975 hasta 1976T2 solo hay datos publicados en papel. Para uso web trazable se recomienda usar INE EPA desde 1976T3 y tratar 1975 como dato historico no homogeneo solo si se extrae de publicacion oficial en papel.
- Estado siguiente: si se incorpora la tasa de paro, documentar metadata completa y decidir si se usa serie trimestral, media anual de cuatro trimestres o ambas.
